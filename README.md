## Api de MetaTrader

### Features

- Puede registrar usuarios en la app.
- Puede iniciar sesión con los usuarios y la sesión se guarda en la base de datos.
- Toma los datos de MetaTrader de los usuarios(login, password) para realizar el CopyTrade.
- Se suscriben los usuarios para poder realizar el copy, los usuarios no suscritos no se les podrá realizar el copyTrade.
- Los pagos activan los usuarios por 30 días.

####Comando para instalar todos los paquetes del proyecto.
````
$  npm i
````
